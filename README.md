# Movie Library — Личная кинотека

**Автор:** Иван Петров (или укажите свои имя и фамилию)

## Описание
GUI-приложение для хранения коллекции фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Реализована фильтрация по жанру и году, данные сохраняются в JSON-файл и автоматически загружаются при запуске.

## Требования
- Python 3.6+
- Стандартные библиотеки: tkinter, json, os

## Установка и запуск
1. Скачайте файл `movie_library.py`
2. Запустите в терминале:
   ```bash
   python movie_library.py
